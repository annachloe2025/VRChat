# 世界観

自作ファンタジー小説の世界観を、VRChatワールド化するための観点で整理する場所。
小説本編の全設定をコピーするのではなく、**ヴィジュアル化に必要な情報**を選別・再編成する。

## このセクションの構成

- [地理](geography.md) — 大陸、地形、気候、ランドマーク
- [歴史](history.md) — 時代区分、重要事件、現在の時系列上の位置
- [文化・種族](cultures.md) — 種族、社会、宗教、衣食住、言語

## 整理の優先順位

VRChatワールドとして実体化することを最終目的にしているので、整理の優先度は次の通り。

1. **空間として体験するときに目に入るもの** — 建築、地形、空、植生、光
2. **空間に意味を与えるもの** — 歴史、文化、住人の暮らし
3. **物語上の設定** — 個別キャラクターの背景、政治的細部 など(後回しでOK)
