# 水回りの仕組み(VR用抜粋)

!!! info "情報源"
    Novel プロジェクト `設定資料/10_生活文化/水回りの仕組み.md` および図解HTMLからの抜粋。
    **VR で表現する際の視覚要素** に焦点を絞ってまとめている。

## 設計思想

> **「川から借りた水は、きれいにして川に返す」**

工業廃水のような深刻な汚染はないが、汚水をそのまま川に戻すことは「借りたものを汚して返す」のと同じ感覚。水守の浄化作業は技術的必要性だけでなく、**文化的に当然の作法** として根付いている。

## 村レベル(50〜100人規模)の水循環

シンプルな一本道の循環:

```
[川上]
  │
  ├─→ 支流取水(土守が開閉) ─→ 浄化の術式石① ─→ 飲み水用ため池(高台)
  │                                                        │ 重力で配水
  │                                                        ▼
  │                                                  [集落: 住居・浴場・畑]
  │                                                        │ 暗渠で排水
  │                                                        ▼
  │                                                  下水用ため池(低地)
  │                                                        │
  │                                                        ▼
  │                                                  浄化の術式石②
  │                                                        │
  │                                                ┌──────┴──────┐
  │                                                ▼              ▼
  │                                            (川下へ戻す)    (汚泥 → 畑の肥料)
  ▼
[川下]
```

## 構成要素(VR で配置するもの)

### 1. 川と支流の取水口

- 村の脇を **南北に流れる川**(または小さな川)
- 川上側に **支流の引き込み口** ―― 土魔法で成型された水路の入口
- 取水口に **木製の堰板** か **土魔法で開閉する仕切り** ―― 水量調整可能
- 取水路は土魔法で固められた浅い溝、幅 1〜1.5m

### 2. 土守の管理小屋(取水口横)

- 土属性の魔石持ちが常駐するわけではない、**作業時だけ来る** 小さな小屋
- 取水量の調整ができる場所、簡単な道具を置ける広さ(2m × 2m 程度)
- 屋根の有無は任意、簡素

### 3. 浄化の術式石①(取水後)

- **紫色の魔石** が中心の小さな浄化装置
- 周囲を土魔法で成型した受け皿/枠(円柱状か、四角い祭壇風)
- 高さ 1〜1.5m、直径 60〜80cm
- 水が術式石を通過すると不純物が沈殿(VR では水の色が薄く澄んだ青に変わる演出)
- 朝、水守が起動する(普段は静かに発光)

### 4. 飲み水用ため池(高台)

- 集落より **2〜4m 高い位置** に配置
- 円形か楕円形、直径 5〜8m、深さ 2m 程度
- 周囲は土魔法で固められた縁(モルタル塗り)
- 蓋はせず、上部は開放(虫が入らないよう浅い屋根 or 格子をかけることもある)
- ここから集落へは **暗渠**(地下水路)で重力配水

### 5. 暗渠(地下水路)

- 集落内では **地表に格子蓋** が見える
- 数メートルおきに点検用のマンホール(土魔法で成型された丸い蓋)
- 各家屋の入口近くに **取水口(蛇口の代わり)** があり、桶で水を汲める
- 浴場・井戸・畑へも分岐

### 6. 集落内の井戸(暗渠の取水口)

- 中央広場の井戸は **暗渠から繋がっている** ―― 地下水を汲んでいるのではない
- 木製の桶と滑車で水を汲み上げる
- これが集落のメインの水源(各家の取水口とは別)

### 7. 下水用ため池(低地)

- 集落より **2〜4m 低い位置** に配置
- 飲み水池より一回り大きい(直径 8〜10m)
- 周囲を土魔法で固めた縁、上部は格子で蓋(蝿対策)
- 集落の暗渠から重力で流れ込む
- ここで一旦汚水と汚泥が分離する

### 8. 浄化の術式石②(下水池の出口)

- 同じく紫色の魔石、構造は①と同じ
- ここを通過した水は **澄んだ緑がかった水** (VR で色変化の演出)
- 夕方、水守が起動

### 9. 川下への放水路

- 浄化された水を川下に戻す浅い溝
- 出口は川岸の少し下流側、自然に合流するように

### 10. 汚泥の搬出路

- 下水池に溜まった汚泥(沈殿物)を **畑の肥料** として運ぶ
- 木製の手押し車か、桶で運搬
- 畑の縁に汚泥置き場(囲いだけの小エリア)

## 水守の日課(VR で再現するなら時間演出)

- **朝**: 術式石① を起動 → 取水と浄化を開始
- **夕**: 術式石② を起動 → 下水浄化
- **適宜**: 汚泥を畑に運ぶ
- **不在時**: 平民数人で代行可能(術式石の起動は意外と簡単)

## VR で表現するキー要素まとめ

| 要素 | 視覚的特徴 | VR での見せ場 |
|------|----------|------------|
| 支流取水口 | 土魔法で成型された水路、堰板 | 入口で水の流れが分岐 |
| 浄化の術式石(2基) | 紫色の魔石、彫像的な構造 | 朝・夕で色合いが変わる演出 |
| 飲み水ため池(高台) | 開放型、青く澄んだ水 | プレイヤーが見下ろせる |
| 暗渠(格子蓋) | 集落内の地表に格子 | 歩いていると気付く小さなディテール |
| 中央井戸 | 暗渠と繋がっている | 木桶と滑車、水音 |
| 下水池(低地) | 格子蓋、やや濁った水 | 集落から下りた先 |
| 川 | 村の脇を流れる | 遠景に水音 |

## VR 化の高低差設計

水システムを表現するには **村に高低差が必要** :

- 飲み水池: 集落から +2〜4m(小高い丘)
- 集落: 平地
- 下水池: 集落から -2〜4m(谷側)

これは VR ワールドの面白さ(歩く中で景色が変わる)にも繋がる。
