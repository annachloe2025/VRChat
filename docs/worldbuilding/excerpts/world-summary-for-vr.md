## 世界の核心(VR で表現すべき空気)

- **魔法のある中世風ファンタジー**
- 魔法の大原則: **創造不可、操作・変換のみ**
- **第三次スタンピードの予兆が迫る現在** ―― 物語の時代背景。日常に静かな緊張感がある
- 人類は「魔力の薄い辺境」に細々と生き延びている
- 魔物の脅威が常にある世界

## 国家の状況(舞台はルクスマーレ王国)

- **ルクスマーレ王国(王都ルメリア)** ―― 東の国
- 他に2国: コンコルディア盟邦、ドミナトゥス帝国
- 3国は対魔物同盟を結んでいる

## 魔法と社会

- **女性は男性の約2倍の魔力量**(副魔石による) ―― 社会的・戦闘的に女性優位
- 魔石持ちは人口の **5〜8%** ―― 希少な人材
- **無魔人**(魔石を持たない者)が多数派
- 魔力属性は **髪色に発現**(無機系=4大属性: 土・水・火・風)
    - 土属性: 茶/土色系
    - 水属性: 青系
    - 火属性: 赤/橙系
    - 風属性: 緑/青緑系
- 有機魔法は髪色に出ない(家系遺伝)

## 建築様式(VR で最重要)

### 大原則

- **個別ブロック(レンガ)を積む文化はない**
- **型枠に素材を詰めて、土魔法で一体成型** が基本
- 壁はなめらか〜彫刻的で、レンガ目地は無い
- **窓ガラスはない** ―― 鎧戸(木板)か格子のみ

### 建材(階層別)

| 場所 | 壁 | 屋根 | 床 |
|------|-----|------|-----|
| 農村 | 圧縮土壁(ベージュ〜茶) | 木板葺き(有機魔法処理) | 土間(土魔法で平らに) |
| 都市 | 焼成粘土壁(オレンジ茶) | 木板葺き(高品質) | モルタル仕上げ |
| 富裕層 | モルタル/コンクリート壁(灰白) | 木板葺き | 磨きモルタル |
| 要塞・王宮 | 分厚いモルタル壁(灰) | モルタル屋根 | モルタル |

### 工法のリアリティ

- **土魔法使い1人いれば、3日で小屋1棟建つ**
- 家を建ててもらうお礼は食料・衣類・労働(物々交換が主流)
- 土魔法使いは村で最も社会的地位の高い職人

## 光源

- **窓ガラスが無い → 屋外の自然光は鎧戸を開けたときだけ入る**
- 屋内は **術式石(magic stones with embedded scripts)** による照明が中心
- 火源としての松明・蝋燭もある(術式石がない庶民の住居)
- 王宮や貴族の家は術式石シャンデリアで青白〜黄白の柔らかい光

## 衣食住の感触

- 衣服: 自然素材中心、染色は植物染め(派手でない)、貴族は装飾的
- 食: 中世風 + 魔法による加熱調理
- 住: 一体成型壁の歪みのない/わずかに有機的な造形

## 時代背景の演出メモ

- 第三次スタンピードが近い ―― 村の見張りが増えている、防壁の補修跡、避難訓練の痕跡
- 家屋には **術式石による封印**(貴族・商人の家)
- 庶民の家は **閂(かんぬき)** 、物理鍵は未発達

## VR 化で **避けるべき** こと

- ✗ 個別レンガが見える壁(積み上げ式)
- ✗ ガラス窓
- ✗ 蛍光色の派手な装飾
- ✗ 十字架等の現代キリスト教モチーフ(独自神話の世界)
- ✗ 西洋ファンタジーの典型(ドラゴン城・派手な紋章)

## VR 化で **積極的に表現したい** こと

- ✓ 一体成型のなめらかな壁、わずかな手作り感
- ✓ 鎧戸の開閉で表情が変わる窓
- ✓ 屋内の柔らかい青白い光(術式石)
- ✓ 防壁・見張り台といった「魔物への備え」の存在感
- ✓ 木の質感(有機魔法処理された板) ―― 防水で硬いがあくまで木
- ✓ 「日常が脅威の隣にある」緊張感
