# VRChat Fantasy World

自作ファンタジー小説の世界観を VRChat ワールドとして再構築する個人プロジェクトのドキュメント母艦。

## このサイトについて

頭の中にしか存在しない世界を、一度ドキュメントに書き出し、最終的には VRChat 上で歩ける空間として実体化する。
そのプロセスの **設計図と日誌** を一つの場所に集約する。

## ナビゲーション

- **[世界観](worldbuilding/index.md)** — 小説本編の設定をVRワールド向けに整理
- **[ヴィジュアル設計](visual-design/index.md)** — ムードボード、建築、自然、色彩の方針
- **[ワールド仕様](world-spec/index.md)** — 範囲、動線、ギミックの設計
- **[制作日誌](progress/index.md)** — 日々の試行錯誤の記録
- **[リファレンス](reference/index.md)** — 技術メモ、アセット一覧
