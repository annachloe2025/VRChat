# ヴィジュアル設計

世界観の **見た目** をどう作るかを言語化するセクション。
小説では曖昧でも問題ない部分(石材の質感、空の色、光の角度など)を、ここで明確化する。

## このセクションの構成

- [ムードボード](moodboard.md) — 全体の雰囲気を決める参考画像群
- [建築様式](architecture.md) — 街・神殿・住居の構造と装飾の方針
- [自然・植生](nature.md) — 樹木、地表、空、水の表現方針

## 進め方の指針

ヴィジュアル設計は「言葉だけで決めようとしない」のがコツ。
参考画像を集める → キーワードで言語化する → Unity でテストアセットを置いてみる、のループを回す。
