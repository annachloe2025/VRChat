# スコープ

## 第1ワールドの方針

**辺境の村** を最初の制作対象とする。詳細設計は [辺境の村](frontier-village.md) を参照。

選定理由:

- 世界観の「縮図」になる(4属性魔石持ち + 無魔人 + 一体成型建築 + 第三次スタンピード前夜の緊張感、すべて自然に現れる)
- 屋外+屋内の両方が学べる(VRChat ワールド制作の基礎技術が一通り入る)
- 失敗のリカバリーが効きやすい(部分修正で済む)

## 想定プレイ時間

5〜10分の散策。明確な目的は無く「世界の中にいる」体験を提供。

## 容量目標

- PC ワールド: 50MB 以内を目標
- Quest 対応: 当面しない(将来検討)

## 公開範囲

- v0.1 〜 v0.5: **Private**(自分とフレンドのみ)
- v1.0 以降: Community Labs を経て Public 検討

## 他の候補(将来用に保留)

- 王宮内(東の国)― 物語のS7舞台。実装難度高め、優先度後
- 主人公の家・拠点 ― 物語の中心。設定固まり次第
- 禁域の入口 ― ダーク系・神秘系
- 公爵家 ― 貴族の暮らし
- 王都ルメリア中心部 ― 大規模、最終目標
