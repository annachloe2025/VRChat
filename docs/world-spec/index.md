# ワールド仕様

VRChatワールドとして実装するときの **技術仕様と範囲** を決めるセクション。
ここを早めに固めると、世界観整理やヴィジュアル設計の優先順位が決まる。

## このセクションの構成

- [スコープ](scope.md) — どの範囲を作るか、容量制約、Quest対応の有無
- [動線・ギミック](interactions.md) — プレイヤーがどう動き、何が起きるか

## VRChat側の制約メモ

- PC ワールド: ファイル容量上限あり(目安として ~100MB 圧縮後)
- Quest 対応する場合は更に厳しい上限・シェーダー制限
- ドローコール、ポリゴン数、テクスチャサイズは早めに方針を決める
